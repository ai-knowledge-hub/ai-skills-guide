# Creative Workshop for PMax + Reels

## When to use
Use for new creative ideation, refresh cycles, and performance-driven variant generation.

## Inputs required
- product_offer
- audience_segment
- campaign_goal
- brand_tone

## Workflow
1. Produce 5 messaging angles.
2. Generate variants for PMax and Reels separately.
3. Validate field lengths and prohibited terms.
4. Return publish-ready structured output.

## Output format
- Angle summary
- Platform-specific creative matrix
- Compliance notes

## Guardrails
- Respect platform character limits.
- Do not include restricted claims.
- Mark uncertain compliance as needs-review.
