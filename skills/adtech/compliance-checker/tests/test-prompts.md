# Test Prompts - Ad-Tech Compliance Checker

1. "Check this Meta ad set copy and landing URL before launch."
Expected: policy and URL checks with severity labels.

2. "Why was this ad disapproved?"
Expected: likely causes mapped to policy-sensitive phrases and landing concerns.

3. "Review these URLs for UTM compliance."
Expected: tracking validation with concrete remediation notes.

4. "Audit this campaign before launch and classify each issue as critical, warning, or info."
Expected: severity-labeled findings and clear launch recommendation.

5. "Rewrite these blocked claims to policy-safer alternatives without changing offer intent."
Expected: compliant rewrites with rationale and explicit caveat that approval is not guaranteed.
