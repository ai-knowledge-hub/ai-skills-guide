# SEO -> Paid Search Synergy

## When to use
Use when aligning SEO insights with paid search planning and expansion.

## Inputs required
- top_queries
- landing_page_performance
- paid_search_constraints

## Workflow
1. Select high-intent SEO winners.
2. Cluster keywords by intent.
3. Propose paid ad groups and starter copy.
4. Output test-ready campaign recommendations.

## Output format
- SEO winners table
- Proposed paid ad groups
- Keyword + ad copy starter set
- Launch experiment plan

## Guardrails
- Separate branded vs non-branded intent.
- Mark speculative mappings as hypotheses.
- Do not assume conversion intent without evidence.
