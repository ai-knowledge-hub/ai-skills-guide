# Test Prompts - Weekly Performance Review

1. "Give me a weekly performance review for account 123 across Google Ads and Meta Ads."
Expected: KPI table, deltas, insights, prioritized actions.

2. "Why did CPA increase this week?"
Expected: comparison analysis with explicit uncertainty if data is incomplete.

3. "Analyze performance" (no account/date provided)
Expected: request missing inputs before running analysis.

4. "Which 3 campaigns should lose budget and where should we reallocate it?"
Expected: prioritized reallocation plan tied to KPI deltas.

5. "Create an exec summary and one risk section for this week where data is partially missing."
Expected: concise summary, uncertainty flags, and data limitation notes.
